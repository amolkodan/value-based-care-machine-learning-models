## Roadmap

### Phase 1: Open-source foundation

- Improve project positioning and contribution docs.
- Standardize model metadata and report artifacts.

### Phase 2: Specialized model suite

- Expand risk and cost model families.
- Add temporal and uplift-oriented modeling flows.

### Phase 3: Evaluation and governance

- Add calibration and subgroup fairness slices.
- Generate leaderboard outputs and model cards.

### Phase 4: Policy and agentic workflows

- Add constrained budget policy simulation.
- Add explainable agentic orchestration for recommendation pipelines.

### Phase 5: Optional scale-up track

- Add cloud-agnostic MLflow-compatible logging patterns.
- Provide optional Databricks-aligned lakehouse templates.
